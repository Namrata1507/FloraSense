How FloraSense Works

Our deep learning pipeline simplifies species identification.

1. Image Input

User uploads a high-resolution plant image.

2. Preprocessing

Image is resized and normalized for the AI model.

3. CNN Identification

Our trained neural network predicts the species.

4. Classification

Database maps the species to 'Non-Invasive' or 'Invasive'.
